## Project Overview
Interactive web-based visualizations exploring two decades of global renewable energy expansion (1990-2023).

## Project Structure
```
Project Skeleton/
├── index.html          # Main HTML page
├── styles.css          # Styling
├── js/

├── data/              # Data files (to be added)
└── README.md          # This file
```

## Planned Visualizations
1. **Choropleth World Map** - Global renewable energy share by country
2. **Multi-Country Line Chart** - Trend comparison over time
3. **Stacked Area Chart** - Energy mix by source (solar, wind, hydro)
4. **Diverging Bar Chart** - Year-over-year changes
5. **Scatter Plot** - Economic context (GDP vs renewable share)
6. **Radial Flower Glyphs** - Innovative energy composition view
